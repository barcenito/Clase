ACTIVIDAD 10. APARCAMIENTO. Se tiene un aparcamiento con 50 plazas, numeradas de 1 a 50, al que intentan acceder coches continuamente. La entrada de coches se puede simular con el programa principal que, cada cierto tiempo (aleatorio entre 500 y 3000ms) crea un hilo que simula un coche. Cada coche se crea con una matrícula distinta. Cada coche intenta conseguir acceso al aparcamiento, indicando su matrícula para ello, que queda registrada para la plaza de aparcamiento libre, si la hay, que se le asigna. Pasado un tiempo aleatorio (por ejemplo 10 a 20s), el coche abandona el aparcamiento. Desarrolla un programa que simule el parking.
